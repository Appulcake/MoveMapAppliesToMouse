# MoveMapAppliesToMouse

Tiny helper mod that allows you to choose which input pair the drag click move map uses, available options:
- Pan View / Tilt View (default)
- Move Lateral / Move Longitudinal
- Move Map Horizontal / Move Map Vertical (note that these conflict with the default joystick/non-mouse axis inputs doing the same without drag-click, that mode is disabled when you set it to this mode, making this set of axis input essentially mouse-only)
